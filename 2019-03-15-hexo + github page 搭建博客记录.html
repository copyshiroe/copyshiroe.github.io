hexo + github page 搭建博客记录

2019-03-15
教程
Hexo/博客

配置 github 仓库

  1. 创建一个新的仓库，名字为用户名.github.io。
  2. 下载 git 客户端安装 https://git-scm.com/download/ 或 https://git-scm.com/download/gui/windows ，配置好账号密码和 ssh key 即可。
  3. 然后打开 cmd git clone https://github.com/用户名/用户名.github.io.git，下载仓库到你喜欢的位置存放。

配置 Hexo

  1. 安装Node.js。
  2. 使用 cmd npm install -g hexo-cli 安装 Hexo 。
  3. 在用户名.github.io文件夹下使用 hexo init 初始化 Hexo 。
  4. npm install
  5. hexo server或hexo s启动本地服务器，默认在 4000 端口。
  6. 在 themes 文件夹里使用git clone https://github.com/lotabout/very-simple.git下载very-simple，在 _config.yml 文件更改 theme 为 very-simple。

写文章

  1. 创建一篇新文章hexo new [layout] <title>，layout 可选，默认为 post ，可以通过修改 _config.yml 中的 default_layout 参数来指定默认布局。
  2. Front-matter 预先定义的参数：
参数 描述 默认值
layout 布局
title 标题
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
comments 开启文章的评论功能 true
tags 标签（不适用于分页）
categories 分类（不适用于分页）
permalink 覆盖文章网址

部署 Hexo 到 Github

  1. 安装 hexo-deployer-git。npm install hexo-deployer-git --save
  2. 修改根目录下 _config.yml，添加下面内容：
  1. 清除站点文件hexo clean。
  2. 生成站点文件hexo generate或hexo g。
  3. 监视文件变动hexo generate --watch。
  4. 部署文件hexo deploy或hexo d。
  5. 完成后部署hexo generate --deploy或hexo deploy --generate，简写为hexo g -d或hexo d -g。
