1012. 十进制整数的补码

题目描述

每个非负整数 N 都有其二进制表示。例如， 5 可以被表示为二进制 "101"，11 可以用二进制 "1011" 表示，依此类推。注意，除 N = 0 外，任何二进制表示中都不含前导零。

二进制的补码表示是将每个 1 改为 0 且每个 0 变为 1。例如，二进制数 "101" 的二进制补码为 "010"。

给定十进制数 N，返回其二进制表示的补码所对应的十进制整数。

  1. 0 <= N < 10^9

思路和答案：

  1. 先十进制转二进制，然后反转每一位，再二进制转十进制。需要注意本题中 0 的补码为 1。时间复杂度为$O(n)$,空间复杂度为 $O(1)$。

  2. 排名第一 JOHNKRAM 的方法：根据题目的规律可以得出，原码 + 补码 = 1111...这样每个二进制位都为 1 的形式；而 2^n^ 的二进制形式为 1000...的形式，减去 1 则变成 1111...的形式；所以 $补码 = 2^n - 1 - 原码$ （ $2^n > N$）。时间复杂度为$O(n)$,空间复杂度为 $O(1)$。
